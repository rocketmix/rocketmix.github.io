A propos de RocketMiX

Vous souhaitez échanger des données de toutes sortes avec vos entreprises partenaires ? Vous en avez assez des interfaces informatiques avec des délais de traitement ? RocketMiX est la plateforme de publication d'API, gratuite et open source, destinée aux entreprises qui souhaitent tourner leur SI vers l'extérieur le plus simplement possible !

En quelques mots, RocketMiX, c'est :

  • Une soluiion d'APIsation pour échanger vos données
  • Une architecture orientées microservices
  • Une intégration ultra simplifiée et prépackagée de produits reconnus tels que Netflix© Zuul, Eureka, Ribbon, Spring Boot Admin, et Swagger UI
  • Une installation réduite au strict minimum (juste un éxecutable Linux et rien d'autre!
  • Un portail utilisateur (avec génération automatique de la documentation), un bac à sable de test des API, une interface graphique d'administration
  • Une configuration "zéro paramétrage"
  • Un kit de démarrage "Spring Boot Starter" pour vous aider à construire vos propres API
  • Une integration native Linux pour vous permettre installer vos API en tant que service Linux (systemd)

Besoin d'une démo ? Testez RocketMix grâce à Heroku. (Identifiez-vous avec admin/admin)

5 minutes pour démarrer!

Vous avez une machine Linux 64 bits sous la main avec 2Go de Ram et 500Mo de disque sur disponible ? OK, vous êtes prêt!

En pré-requis, vous devez installer un JDK 8 ou supérieur. Faites
  • sudo yum install java-1.8.0-openjdk-devel sous CentOS
  • ou sudo apt install default-jdk sous Debian et Ubuntu

Téléchargez et lancez l'éxecutable Linux (une seule ligne de commande suffit!) :


wget https://sourceforge.net/projects/rocketmix/files/1.0.43-SNAPSHOT/rocketmix-routing-server-1.0.43-SNAPSHOT.war/download -O rocketmix-routing-server.war && chmod +x rocketmix-routing-server.war && export PID_FOLDER=$(pwd) && export LOG_FOLDER=$(pwd) && ./rocketmix-routing-server.war run

Votre serveur API démarre sur le port 8080 (http://localhost:8080). Ouvrez un navigateur internet pour accéder à votre portail web. Consultez votre catalogue d'API et accéder à votre interface d'administration (le compte est admin/admin par défaut).

En production, vous pouvez installer votre serveur (et vos APIs) sous forme de service Linux. Lancez rocketmix-routing-server.war install pour déployer votre serveur en tant que service Linux (systemd). Respirez : systemd est sécurisé et vous garantit le redémarrage de votre serveur en cas de panne. Avec RocketMiX et Linux, vous pouvez dormir l'esprit tranquille.

Développez vos propres API (en un temps record)

Vous êtes développeur Java et vous connaissez Spring et Maven ? Ca tombe bien!

On vous a préparé un module Spring Boot pour simplifier le lancement de votre projet. Importez l'artifact suivant dans votre pom.xml :

<!-- https://mvnrepository.com/artifact/io.github.rocketmix/rocketmix-spring-boot-starter -->
<dependency>
    <groupId>io.github.rocketmix</groupId>
    <artifactId>rocketmix-spring-boot-starter</artifactId>
    <version>1.0.43-SNAPSHOT</version>
</dependency>

A partir de là, le plus simple est d'aller consulter le projet de démo sur github.
